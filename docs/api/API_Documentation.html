📋 סקירה כללית

CartWise Pro API היא REST API המספקת ממשק מלא לניהול מערכת השכרת עגלות קניות חכמות.

🌐 Base URL

http://localhost:8002

מאפיינים עיקריים

  • ✅ אימות דו-שלבי (OTP) באמצעות SMS
  • ✅ מערכת טוקנים לאימות מתמשך (30 יום)
  • ✅ ניהול עגלות ומנעולים אלקטרוניים
  • ✅ מעקב אחר השכרות פעילות
  • ✅ התראות SMS אוטומטיות
  • ✅ תמיכה ב-RS485 לבקרי CU16

🚀 Quick Start

// 1. Request OTP
fetch('http://localhost:8002/auth/request-otp', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({phone: '0501234567'})
})

// 2. Verify OTP & Get Token
fetch('http://localhost:8002/auth/verify-otp', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({phone: '0501234567', otp_code: '1234'})
})

// 3. Use Token for Cart Operations
fetch('http://localhost:8002/carts/assign', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN'
  },
  body: JSON.stringify({phone: '0501234567', otp_code: '0000'})
})

🔐 אימות (Authentication)

המערכת משתמשת באימות דו-שלבי עם טוכנים לשימוש חוזר.

שלב ראשון: OTP חד-פעמי (פעם אחת)
שלב שני: טוקן תקף ל-30 יום (לשימוש חוזר)
POST /auth/request-otp

תיאור: שלח קוד OTP ל-SMS

Request Body:

{
  "phone": "0501234567"
}

Response (200 OK):

{
  "success": true,
  "message": "קוד אימות נשלח בהצלחה",
  "expires_in_minutes": 5,
  "already_authenticated": false
}
💡 טיפ: אם שולחים טוקן ב-Authorization header והוא תקף, המערכת תדלג על שליחת SMS!
POST /auth/verify-otp

תיאור: אמת קוד OTP וקבל טוקן אימות

Request Body:

{
  "phone": "0501234567",
  "otp_code": "1234"
}

Response (200 OK):

{
  "success": true,
  "message": "קוד אומת בהצלחה",
  "phone": "0501234567",
  "auth_token": "abc123...",
  "token_expires_days": 30
}
✅ שמור את הטוקן!

⚠️ קודי שגיאה

שגיאות נפוצות

Code שגיאה פתרון
401 קוד שגוי או פג תוקף בקש OTP חדש
400 יש לך כבר עגלה פעילה החזר את העגלה הנוכחית
404 אין עגלות זמינות המתן להחזרת עגלה
500 שגיאת שרת נסה שוב מאוחר יותר

⚠️ טיפול בשגיאות

try {
  const response = await fetch('/carts/assign', {...});
  const data = await response.json();

  if (!response.ok) {
    switch (response.status) {
      case 400:
        alert('יש לך עגלה פעילה');
        break;
      case 401:
        alert('קוד שגוי');
        break;
      case 404:
        alert('אין עגלות זמינות');
        break;
    }
    throw new Error(data.detail);
  }
} catch (error) {
  console.error(error);
}

💡 דוגמאות קוד

דוגמה מלאה - JavaScript

// Complete user flow
async function completeFlow() {
  const phone = '0501234567';

  // 1. Request OTP
  await fetch('/auth/request-otp', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({phone})
  });

  console.log('📱 Check SMS for OTP');
  const otp = prompt('Enter OTP:');

  // 2. Verify OTP & Get Token
  const verifyRes = await fetch('/auth/verify-otp', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({phone, otp_code: otp})
  });

  const {auth_token} = await verifyRes.json();
  localStorage.setItem('cartwise_auth_token', auth_token);

  // 3. Assign Cart
  const assignRes = await fetch('/carts/assign', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${auth_token}`
    },
    body: JSON.stringify({phone, otp_code: '0000'})
  });

  const cart = await assignRes.json();
  console.log(`✅ Cart ${cart.cart.cart_id} assigned!`);

  // 4. Return Cart
  await fetch('/carts/initiate-return', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({phone})
  });

  await fetch('/carts/complete-return', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({phone})
  });

  console.log('✅ Cart returned!');
}

completeFlow();

דוגמה - Python

import requests

class CartWiseClient:
    def __init__(self, base_url='http://localhost:8002'):
        self.base_url = base_url
        self.token = None

    def request_otp(self, phone):
        response = requests.post(
            f'{self.base_url}/auth/request-otp',
            json={'phone': phone}
        )
        return response.json()

    def verify_otp(self, phone, otp_code):
        response = requests.post(
            f'{self.base_url}/auth/verify-otp',
            json={'phone': phone, 'otp_code': otp_code}
        )
        data = response.json()
        self.token = data['auth_token']
        return data

    def assign_cart(self, phone):
        response = requests.post(
            f'{self.base_url}/carts/assign',
            headers={'Authorization': f'Bearer {self.token}'},
            json={'phone': phone, 'otp_code': '0000'}
        )
        return response.json()

# Usage
client = CartWiseClient()
client.request_otp('0501234567')
client.verify_otp('0501234567', '1234')
cart = client.assign_cart('0501234567')
print(f"Cart {cart['cart']['cart_id']} assigned!")
